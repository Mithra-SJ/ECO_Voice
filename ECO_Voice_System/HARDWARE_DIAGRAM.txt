Power Rails:
  Red   (+) ──── 5V from ESP32-S3
  Blue  (-) ──── GND (Common Ground)

Component Placement:
  Row 1-10:   ESP32-S3 Dev Module
  Row 15-20:  INMP441 Microphone
  Row 25-30:  INA219 Current Sensor
  Row 35-40:  PIR Sensor (or external mount)
  Row 45-50:  LDR Module
  Row 55-60:  Relay Module (or external due to size)

Tips:
  - Use color-coded wires (Red=Power, Black=GND, Others=Signal)
  - Keep I2C wires short (<15cm) to reduce noise
  - Separate power-heavy components from sensitive analog sensors
  - Add 0.1µF capacitors near sensor VCC pins for noise filtering


┌─────────────────────────────────────────────────────────────────────────────┐
│                            SAFETY CHECKLIST                                 │
└─────────────────────────────────────────────────────────────────────────────┘

Before Powering On:
  ☐ All GND connections are common
  ☐ No short circuits between VCC and GND
  ☐ 3.3V components not connected to 5V
  ☐ I2S pins correctly mapped (SCK, WS, SD)
  ☐ I2C pull-up resistors present (usually on module)
  ☐ Relay module properly isolated from logic
  ☐ Load current within relay rating (10A max)
  ☐ Polarities correct for LEDs
  ☐ All wire connections secure

AC Power Warning (if using):
  ⚠ DANGER: Lethal voltage! Only if qualified
  ⚠ Use proper enclosures and insulation
  ⚠ Add circuit breakers and fuses
  ⚠ Never work on live AC circuits
  ⚠ Test with low voltage DC loads first

Signal Integrity:
  ☐ I2S data lines away from power traces
  ☐ Ground plane on breadboard or PCB
  ☐ Decoupling capacitors on sensor power pins
  ☐ Twisted pair for long I2C wires (optional)


┌─────────────────────────────────────────────────────────────────────────────┐
│                          COMPONENT SPECIFICATIONS                           │
└─────────────────────────────────────────────────────────────────────────────┘

ESP32-S3:
  - Voltage: 3.3V logic, 5V USB power
  - Current: ~500mA (peak)
  - Flash: 8MB
  - PSRAM: 8MB (OPI)

INMP441:
  - Voltage: 1.8V - 3.3V
  - Current: <1mA
  - Sensitivity: -26dBFS
  - SNR: 61dB

PIR HC-SR501:
  - Voltage: 4.5V - 20V
  - Current: <50µA standby
  - Range: 3-7m
  - Delay: 0.3s - 200s (adjustable)

INA219:
  - Voltage: 3.3V - 5V (logic), 0-26V (measured)
  - Current: ±3.2A max
  - Resolution: 0.8mA
  - I2C Address: 0x40 (default)

LDR Module:
  - Voltage: 3.3V - 5V
  - Resistance: 1KΩ (bright) - 10MΩ (dark)
  - Response Time: ~20ms

Relay Module (2-Channel):
  - Voltage: 5V (coil)
  - Current: 15-20mA per channel
  - Contact Rating: 10A @ 250VAC / 30VDC
  - Isolation: Optocoupler
